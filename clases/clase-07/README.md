# Clase 07

- Discutimos sobre lo encontrado de hambrehambrehambre

- Ejercicio en clase: Los estudiantes deben elegir 3 objetos de su interés, seleccionar una carta de la baraja de “Estrategias oblicuas” para crear una frase y describir una relación entre ellos, para luego proponer una tabla de información con una selección de tiempo y momento.
    
- “Estrategias oblicuas” de Brian Eno y Peter Schmidt es una baraja de cartas con frases que promueven la creatividad.

- Web (en inglés) para ver las estrategias oblicuas <https://obliquestrategies.ca/>
